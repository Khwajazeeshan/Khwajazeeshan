### 🚀 Featured Projects

* [**Revamping UAJK Portal Empowered with AI Chatbot**](https://uajk-portal.vercel.app/)
  Built a full MERN-based university portal with a modern UI, 3D model integration. Integrated AI assistant for smart recommendations. Added an Admin Dashboard for student tracking and a Student Dashboard for registrations and profile management. Focused on smoother user experience and automated support.

* [**SecureAuth**](https://secure-auth-ten.vercel.app/)
  A full-stack authentication system using Next.js, MongoDB, and Tailwind CSS with JWT access/refresh token architecture. Implemented OAuth login (Google, GitHub), email verification, password recovery, and middleware-based route protection.

* [**Bitlink**](https://bitlink-io.vercel.app/)
  A URL shortening platform using Next.js, Typescript, MongoDB, and Tailwind CSS to convert long URLs into short, shareable links. Implemented custom slugs, dynamic redirection routes, and a responsive UI for fast and ad-free link generation.

* [**Premium Task Workspace**](https://task-workspace.vercel.app/)
  Task management application using Next.js, Typescript, and MongoDB with passwordless magic-link authentication and JWT session security. Implemented full task CRUD operations, real-time productivity statistics dashboard, and responsive UI using Tailwind CSS.

* [**LinkNest**](https://link-nest-rho.vercel.app/)
  A link-in-bio platform using Next.js, MongoDB, and Tailwind CSS enabling users to manage and share multiple social links via a single profile page. Implemented JWT-based authentication, email verification, password reset, and a dashboard for adding, updating, and deleting links.

* [**AI Chatbot**](https://chatbot-omega-snowy.vercel.app/)
  Developed a Web Application for real-time conversational responses using React, Node.js, Express.js, and MongoDB with a microservices-based backend architecture for authentication, chat processing, and user management. Implemented secure authentication with JWT, Email OTP verification, and Google OAuth.
